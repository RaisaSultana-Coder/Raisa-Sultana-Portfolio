Phishing Detection Through Hybrid Machine Learning Based On URL

Phishing attacks are a major cybersecurity threat where attackers impersonate legitimate websites to steal sensitive user information such as login credentials, financial data, and personal details. Traditional blacklist-based detection methods are often ineffective against newly generated phishing sites. This project aims to develop a more robust and intelligent phishing detection system using a hybrid machine learning model that analyzes URL-based features.

Improved Trust In Human Collaboration with Chatgpt

Incorporating ChatGPT into human-robot collaboration (HRC) systems can significantly improve trust between humans and robots, making them more effective collaborators. This is primarily due to ChatGPT's ability to facilitate better communication, understand human language nuances, and build a more intuitive and natural interaction. Studies have shown that using ChatGPT in robots allows humans to control them using natural language, leading to increased trust and reduced cognitive load.
